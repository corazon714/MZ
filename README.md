# MZ ETZ 251 — Motor Bilgi Tabanı

Bu bilgi tabanı, **MZ ETZ 251** motosikletinin motoru (EM 251) hakkında kapsamlı teknik bilgi, bakım prosedürleri, tamir kılavuzları ve sorun giderme rehberlerini içerir.

> **MZ ETZ 251** — VEB Motorradwerk Zschopau (Zschopau Motor Fabrikası) tarafından Ocak 1989 – Aralık 1991 yılları arasında üretilen, tek silindirli, 2 zamanlı, 243 cm³ hacimli hava soğutmalı motora sahip bir motosiklettir.

## Hızlı Referans

| Özellik | Değer |
|---|---|
| **Motor Tipi** | EM 251, tek silindir, 2 zamanlı |
| **Silindir Hacmi** | 243 cm³ |
| **Çap × Strok** | 69 mm × 65 mm |
| **Maksimum Güç** | ~21 PS (15,4 kW) @ 5200 d/d |
| **Maksimum Tork** | ~22,5 Nm @ 4800 d/d |
| **Şanzıman** | 5 ileri, ayakla kumandalı |
| **Karbüratör** | BVF 30 N 3-1 |
| **Ateşleme** | Elektronik (önceki modellerde mekanik platinli) |
| **Elektrik** | 12V, AC jeneratör 180W |
| **Yakıt Sistemi** | Ön-karışım (1:50) veya Getrenntschmierung (ayrı yağlama) |
| **Soğutma** | Hava soğutmalı |
| **Ön Fren** | Hidrolik disk fren (standart); kampana fren (bazı varyantlar) |
| **Üretim Yılları** | Ocak 1989 – Aralık 1991 |
