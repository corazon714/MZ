# Güç Kaybı — Arıza Tespit Rehberi

## Güç Kaybı Türleri

| Tür | Açıklama |
|---|---|
| **Kademeli** | Zamanla yavaşça güç azalıyor |
| **Ani** | Bir anda belirgin güç kaybı |
| **Devire bağlı** | Belirli devir aralığında güç kaybı |
| **Sıcaklığa bağlı** | Motor ısındığında veya soğukta kötü |
| **Yüke bağlı** | Yokuş/rüzgâr/sepet altında kötü |

---

## Kademeli Güç Kaybı

### En Yaygın Nedenler

| Neden | Kontrol | Çözüm |
|---|---|---|
| **Hava filtresi tıkalı** | Filtre görsel/ışık kontrolü | Temizle veya değiştir |
| **Egzoz karbon birikimi** | Motor eski performansın altında | Egzoz iç temizliği |
| **Segman aşınması** | Sıkıştırma testi | Üst revizyon |
| **Buji eskimiş** | Buji rengi, elektrot aralığı | Yeni buji |
| **Ateşleme zamanlaması kayma** (mekanik) | Zamanlama kontrolü | Yeniden ayar |
| **Debriyaj kayması** | Devir artıyor ama hız artmıyor | Debriyaj tamiri |
| **Zincir aşınması** | Aşırı sarkma, düzensiz gerginlik | Zincir + dişli değişimi |
| **Arka fren sürtme** | Tekerleği döndür — ağır mı? | Fren ayarı |

## Ani Güç Kaybı

### Olası Nedenler

| Neden | Ek Belirti | Çözüm |
|---|---|---|
| **Buji arızası** | Düzensiz çalışma, ateşleme kaçırma | Buji değiştir |
| **Yakıt bitme (reserve)** | Devir dalgalanması | Reserve konumuna al |
| **Emme borusu çatlağı/düşmesi** | Hava kaçağı sesi, fakir çalışma | Emme borusu kontrol |
| **Egzoz başlığı gevşeme** | Cızırtılı ses, is sızıntısı | Cıvataları sık |
| **CDI/ateşleme arızası** | Düzensiz kıvılcım | Ateşleme sistemi kontrol |
| **Piston segman kırılması** | Metalik ses, ani güç düşüşü | Motor sök, kontrol et |
| **Yakıt musluk kısmi tıkanma** | Devri yükseltince kötüleşir | Musluğu temizle |

## Sıcaklığa Bağlı Sorunlar

### Motor Isındığında Güç Kaybediyor
| Neden | Çözüm |
|---|---|
| Karışım çok fakir → ısınmayla daha da fakirleşir | Karbüratör ayarı (zenginleştir) |
| CDI ısıya duyarlı arıza | CDI değiştir |
| Ateşleme bobini ısıya duyarlı | Bobin değiştir |
| Aşırı ısınma → pre-ignition başlangıcı | Soğutma sistemi kontrol, buji ısı derecesi |
| Karter simeri ısıyla yumuşayıp sızıyor | Simer değiştir |

### Motor Soğukken Kötü
| Neden | Çözüm |
|---|---|
| Jikle çalışmıyor | Jikle mekanizmasını kontrol |
| Karışım çok fakir (soğuk hava) | Kış ayarı (iğne/jikle) |
| Şanzıman yağı çok koyu | Uygun viskozite yağa geç |

## Mekanik Güç Kaybı

| Neden | Belirti | Kontrol |
|---|---|---|
| **Debriyaj kayması** | Devir artar, hız artmaz | Yokuşta tam gazda test |
| **Zincir aşınması** | Ses, titreşim, boş çekiş | Zincir sarkması ve uzaması kontrol |
| **Arka fren sürtme** | Motosiklet ağır geliyor | Arka tekerleği döndür |
| **Ön fren kaliper sıkışma** | Düz yolda yavaşlama | Ön tekerleği döndür |
| **Lastik basıncı düşük** | Ağır sürüş hissi | Basınç kontrol |
| **Yatak arızası** | Hırıltı sesi | Tekerlek yatakları kontrol |

---

## Güç Kaybı Test Prosedürü

1. **Kolay kontroller**: Hava filtresi, buji, lastik basınçları, yakıt kalitesi
2. **Sıkıştırma testi**: <6 bar → üst kısım sorunu
3. **Buji rengi okuma**: Fakir/zengin/ideal
4. **Egzoz kontrolü**: Aşırı duman? Renk? Koku?
5. **Karbüratör kontrolü**: Rölanti ayarı, jikle boyutları
6. **Ateşleme kontrolü**: Kıvılcım kalitesi, zamanlama
7. **Mekanik kontroller**: Fren sürtme, debriyaj, zincir
